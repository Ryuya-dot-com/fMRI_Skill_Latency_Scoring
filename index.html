採点基準 / Scoring Instructions

1. 採点の流れ

  1. Rater ID に採点者 ID を入力する。
  2. 採点する参加者 ID を選択して Start Scoring を押す。
  3. Space キーで音声を再生・停止する。
  4. Accuracy の NR / 0 / 1 で採点する。
  5. 発話の開始点を赤い Onset 線でセットする。
  6. 発話の終了点を青い Offset 線でセットする。
  7. 特記事項があれば Notes に記載する。
  8. 右矢印または Enter で次の試行に進む。前の試行は左矢印。
  9. 全試行が終わったら画面下の Export All (CSV) をクリックする。

2. 発話タイミング

まず発話回数を選択します。1回がデフォルトで、2回以上を選ぶと U1, U2, U3, U4 の発話開始線を追加できます。

赤い線は解析に使う Onset、青い線は Offset、U 線は複数発話や False Start の開始位置です。

False Start（例: gust.. Gustiando）がある場合は、U1 を最初の発声に置き、赤い Onset は最後の本回答に置いてください。

2度完全な単語の発声があった場合は、赤い Onset は1回目の完全な単語の開始点に置いてください。

操作方法:

  • Confirm (C): 自動検出が正しい場合、そのまま承認
  • Correct (drag): マーカーをドラッグして位置を修正
  • Manual Set: 波形をクリックしてオンセットを手動設定
  • U1/U2 Click: 波形をクリックして各発話の開始位置を設定
  • Offset Click (O): 波形をクリックして発話終了位置を設定
  • 無発話 (T): 録音中に一切の発話がない場合
  • 機器不良 (F): 機材トラブルにより録音不良の場合
  • 非語彙音 (G): 咳・息・舌打ちなど非語彙音のみの場合

3. 正確性の採点 (Accuracy)

1 正確: 正しいスプラング語の進行形が産出されている
0 不正確: 間違った形、途中で止めた、異なる単語を産出した場合
NR 無回答: 録音に発話が含まれていない場合（無発話 / 機器不良 / 非語彙音）

採点基準:

  • 細かい音素・音韻単位での違い（例: [r]-[l], [t]-[tʃ], [s]-[ʃ]）は考慮せず、活用カテゴリーが合っていれば正答にする。
  • 細かい音素・音韻の抜けがあっても正答にする。例: partiondo が pationdo と発音された場合。
  • 明らかに違う単語や正答とかけ離れているものは不正答にする。
  • 語尾は聞きづらい場合があるため、語尾が曖昧でも活用が合っていれば正答にする。例: sontiondo が sontion と聞こえる場合。

Double Answer:

  • 発話が2回あった場合は Double answer で DA_FC / DA_SC / DA_IC を選択する。
  • DA_FC は1回目が正答、DA_SC は2回目が正答、DA_IC はどちらも不正答。
  • DA_FC と DA_SC は Accuracy = 1、DA_IC は Accuracy = 0 として扱う。
